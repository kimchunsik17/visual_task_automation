# 🚀 Mock API Server (MVP 해커톤 시연용)

B2B SaaS 및 노코드 비주얼 자동화(iPaaS) 플랫폼 MVP 데모 시연을 위해 제작된 완벽한 Mock API 서버입니다. 
사업자 등록 이슈로 실제 연동이 불가능한 네이버 커머스 및 카카오 알림톡 실서버 환경을 **로컬에서 100% 동일하게 모방(Mocking)** 합니다.

## ✨ 주요 기능

1. **네이버 스마트스토어 주문 발생 트리거기 (Webhook Emitter)**
   - `POST /mock/naver/trigger-order`
   - 실제 네이버 커머스 API의 '주문 내역(ProductOrder)' 규격과 100% 동일한 구조의 더미 JSON 데이터를 생성합니다.
   - 무작위 Latency (500~1500ms) 지연 후 타겟 백엔드 웹훅 URL로 데이터를 전송하여 실제와 매우 유사한 트래픽 환경을 연출합니다.

2. **카카오 알림톡 수신 서버 (Mock Action Server)**
   - `POST /mock/kakao/alimtalk`
   - 알리고/솔라피 등 실제 알림톡 대행사의 API 스펙을 모방하여 데이터를 받고 검증합니다.
   - 전송 성공 시 터미널 콘솔에 **컬러풀하고 직관적인 성공 로그**를 출력하여 심사위원들에게 명확한 결과를 어필할 수 있습니다.

3. **심사위원 시연용 미니 대시보드 (Admin UI)**
   - `GET /`
   - 브라우저 환경에서 타겟 URL을 입력하고 클릭 한 번으로 주문을 발생시킬 수 있는 시연용 UI 대시보드를 제공합니다.

---

## 💻 실행 방법

본 서버는 기존 프로젝트의 프론트엔드(`5173`) 및 메인 백엔드(`8000`) 포트와의 충돌을 피하기 위해 **포트 `3001`**을 사용합니다.

### 1. 패키지 설치
처음 실행 시 필요한 라이브러리를 설치합니다.
```bash
npm install
```

### 2. 서버 실행
터미널을 열고 아래 명령어로 서버를 가동합니다.
```bash
npm start
# 또는 개발 모드: npm run dev
```

### 3. 데모 시연하기
1. 서버 구동 후 브라우저를 열어 **[http://localhost:3001](http://localhost:3001)** 에 접속합니다.
2. 입력창에 테스트할 메인 프로젝트의 웹훅 수신 주소(예: `http://localhost:8000/webhook/1234...`)를 입력합니다.
3. **[스마트스토어 가짜 주문 발생시키기]** 버튼을 클릭합니다.
4. 백엔드 워크플로우를 거쳐 최종 카카오톡 노드가 실행될 때, 이 서버의 카카오톡 수신 엔드포인트(`http://localhost:3001/mock/kakao/alimtalk`)를 호출하도록 연동해 둡니다.
5. Mock 서버 터미널에 🟩 초록색의 멋진 **알림톡 발송 성공 로그**가 출력되는 것을 확인합니다!

## 🛠️ 사용된 기술 스택
- **Node.js / Express**: 경량화된 백엔드 서버
- **Axios**: 타겟 웹훅 URL로 데이터를 전송하기 위한 HTTP 클라이언트
- **Chalk**: 터미널 출력에 다채로운 색상을 부여하여 가독성 높은 시연 연출
- **CORS**: 프론트엔드 UI와의 통신을 위한 크로스 도메인 설정
